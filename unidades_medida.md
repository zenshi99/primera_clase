# 📏 Unidades de medida en diseño web

En CSS usamos **unidades de medida** para definir tamaños, márgenes, fuentes y espacios.  
Estas unidades se dividen en **absolutas** y **relativas**.

---

## ✨ Unidades absolutas
Son fijas y no cambian según la pantalla o dispositivo.  
Se usan cuando quieres un tamaño exacto.

- **px (píxeles)** → la más común, define tamaños exactos.  
- **pt (puntos)** → más usada en impresión.  
- **cm / mm / in** → medidas físicas (centímetros, milímetros, pulgadas), casi no se usan en web.  

👉 Ejemplo:
```css
p {
  font-size: 16px; 
}
